# Taskmaster - Student Planner
### To go along with the flashcard deck application (available in this repository: https://github.com/parkerjacob/FlashcardDeck) I wanted to create an application that allowed students to create a to-do list to keep up with busy class schedules.
# The Goal
### The primary objective of this planner was to create a list of tasks and have them stored under a specific login. This would allow the user to create multiple planners if necessary. I wanted the addition of tasks to be simple, allow the user to select the date and what the course was along with a description.
### The tasks would be viewed in order by the soonest to-do date. Once a task is complete, the user just had to mark as complete to remove it from the list.
# Features Implemented
## Add & Remove
### In the application, you can add tasks as well as remove them with the click of a button. 
### Also, the tasks are user specific. Tasks are presented depending on the user associated with those tasks.
### Overall, this application is user-friendly and useful, especially alongside the flashcard deck listed above.
